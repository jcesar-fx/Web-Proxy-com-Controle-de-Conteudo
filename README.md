# Web-Proxy-com-Controle-de-Conte-do
Web proxy HTTP que atua como intermediário entre cliente e internet. Permite três ações: repasse de conteúdo, bloqueio de sites via blocked.json e filtro de palavras via words.json. Registra logs com timestamp, URL e ação. Desenvolvido para a disciplina de Sistemas para Internet 2 (FURG).
